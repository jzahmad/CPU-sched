1)- Go to the directory where you want to execute the program
2)- type the following command in the input prompt:- "gcc A4.c -o A4"
3)- After that type "./A4"
4)- After that run the following command to see the output:- "cat Output.txt"
5)- Make sure you have the "TaskSpec.txt" in the directory.
